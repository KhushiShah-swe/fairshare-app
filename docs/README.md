# 📄 FairShare Documentation

This folder contains all supporting documentation and artifacts for the **FairShare** project, including sprint presentations, feature breakdowns, and demo videos.

---

## 🚀 Sprint Overview

### 🔹 Sprint 1 – Core Foundation

**Key Features:**
- User Authentication (Signup & Login)
- Dashboard with:
  - Net balance
  - Amount owed / to be received
  - Visual balance representation
- Group Management:
  - Create and join groups
  - View group details and members
- Expense Management:
  - Add expenses with details (title, category, amount, participants)
  - Automatic balance calculation

🎥 **Demo Video:**  
https://youtu.be/nV-iAE6gtnw

---

### 🔹 Sprint 2 – Advanced Expense Management

**Key Features:**
- Percentage-based expense splitting
- Split among selected group members
- Receipt upload and viewing
- Improved UI for expense tracking
- Enhanced backend data handling for flexible splits

🎥 **Demo Video:**  
https://youtu.be/c3UXRfElc08 

---

### 🔹 Sprint 3 – Settlement & Payments

**Key Features:**
- Settlement plan generation to minimize transactions
- “Settle Up” functionality for clearing balances
- Zelle payment integration:
  - Add email and phone number
  - Upload and view QR code
- Improved user experience for payment flow

🎥 **Demo Video:**  
https://youtu.be/GIe5kEu0w4Q
